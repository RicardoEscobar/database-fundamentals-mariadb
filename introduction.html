What you need to know

There are no requirements for this book, we'll be working at a beginners level, yet you'll find value at knowing basic programming with any modern programming language. When I mention variables, loops, conditional statements, functions and arrays, you may not feel confortable, if that's the case, you may look into a fundamentals of programming book or course. Even if I don't cover those subjects here in detail, I'm explaining their use in the real of a database, so you don't need to be a programmer to get value from the book.

We are going to use MariaDB as our database engine, if you don't know what that is, don't worry. Most people commit to their first database for a long time, it happened to me, so I think that MariaDB is ideal for learning and keep using it in a professional enviroment. Learning with something easier like Miscorsoft Access or SQLite, may sound like a great idea because it's easy, but most jobs out there are with databases that support more than one user at the time. Whatever you learn with MariaDB, you'll be able to use for years to come, and even translate your MariaDB skills to other SQL relational databases like: MySQL, SQL server, Oracle, Postgres, etc.

It doesn't really matter what operative system you use, Windows, MacOS and Linux, have access to MariaDB installers. The skills you'll learn here will be applicable to whatever app or program your company needs a database for.
